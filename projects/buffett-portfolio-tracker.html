Buffett Portfolio Tracker

Python GitHub Actions SEC EDGAR Data

The Buffett Portfolio Tracker is a data pipeline and dashboard that automatically pulls Berkshire Hathaway's quarterly SEC 13F filings, parses the holdings data, and renders a clean web view of Warren Buffett's current portfolio.

The pipeline runs on GitHub Actions on a scheduled cron job. The Python script hits the EDGAR XBRL API, parses the XML response, and writes the holdings to a static JSON file that the frontend reads directly — no server required.

What I built

  • Python script to fetch and parse 13F XML from SEC EDGAR
  • GitHub Actions workflow running on a quarterly schedule
  • Static HTML/JS dashboard hosted on GitHub Pages
  • Historical position tracking to show changes quarter-over-quarter

What I learned

Building this taught me the difference between scraping and using a structured API. EDGAR's XML format has quirks around CIK identifiers and namespace handling that took significant debugging to work through.
